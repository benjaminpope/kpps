# K2 Planet search

Oxford K2 planet search pipeline. Searches for transit-like signals from the `k2sc`-detrended light curves.

## Installation

    python setup.py install --user

## Usage

    k2search 

## Disclaimer

Functional but very ugly code, will be cleaned up in the near future.
